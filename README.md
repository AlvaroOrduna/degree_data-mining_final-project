# Mineria-Projecto-Final

*Autores:* Carl Rodriguez (@carl12) y Álvaro Orduna (@AlvaroOrduna)

*Descripción:* Proyecto final de la asignatura Minería de Datos (UPNA)

## Estructura

* __data:__     contiene los archivos de train y de test
* __res:__      recursos proporcionados por el profesor
* __results:__  resultados obtenidos en las diferentes ejecuciones
* __src:__      implementación de los diferentes algoritmos y del script principal

## Ejecución

Para ejecutar cualquiera de los algoritmos implementados basta con modificar la variable `algorithm` del archivo `src\main.m` y ejecutar el script.
Los valores aceptados por la variable `algorithm` son los nombres de las carpetas presentes en `src`.

El script __`src\main.m`__ se encarga de __cargar los datos, normalizarlos, ejecutar el algoritmo seleccionado y guardar los resultados__.
